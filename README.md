
# Akinator Game

A Python implementation of the famous Akinator guessing game where the AI tries to read your mind!

## Features

- Interactive web interface
- AI-powered character guessing
- Questions narrowed down based on your answers
- Learn as the game progresses
- Simple yes/no/probably/probably not responses

## Installation

```bash
git clone <Akinator_Game>
cd Akinator-Game
```

## Usage

```bash
python api/server.py
```

Follow the prompts and answer the AI's questions with:
- `yes` / `y`
- `no` / `n`
- `probably` / `p`
- `probably not` / `pn`

## How It Works

The game uses a decision tree algorithm to narrow down possibilities through strategic questioning until it identifies your character.

- Python 3.7+
